# Social Scout Agents

- `social-scout`: mock/read-only/local candidate agent only. It may summarize
  local fixtures and propose source candidates with confidence. It has no real
  social API path, no scrape path, no network path, no provider or connector
  path, and no publishing path.

Real social source enablement requires an approved policy, source allowlists,
audit trail, compliance review, and scoped credentials before any external path
is added.
